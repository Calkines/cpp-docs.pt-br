# <a name="internationalization"></a>Internacionalização

A biblioteca em tempo de execução da Microsoft fornece muitas rotinas que são úteis para criar versões diferentes de um programa para mercados internacionais. Inclui [rotinas relacionadas à localidade](../c-runtime-library/locale.md), rotinas de caracteres largos, rotinas de caracteres de vários bytes e rotinas de texto genérico. Por conveniência, a maioria das rotinas relacionadas à localidade também são categorizadas nessa referência de acordo com as operações executadas. Nesta seção e na referência alfabética, as rotinas de caracteres de vários bytes e as rotinas de caracteres largos são descritas com as equivalentes de caractere de byte único, quando existem.

Também são incluídas as alternativas do operador ISO646.
