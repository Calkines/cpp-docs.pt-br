# <a name="csqrt-csqrtf-csqrtl"></a>csqrt, csqrtf, csqrtl

Recupera a raiz quadrada de um número complexo, com uma ramificação recortada ao longo do eixo real negativo.

## <a name="syntax"></a>Sintaxe

```C
_Dcomplex csqrt(
   _Dcomplex z
);
_Fcomplex csqrt(
   _Fcomplex z
);  // C++ only
_Lcomplex csqrt(
   _Lcomplex z
);  // C++ only
_Fcomplex csqrtf(
   _Fcomplex z
);
_Lcomplex csqrtl(
   _Lcomplex z
);
```

### <a name="parameters"></a>Parâmetros

*z*<br/>
Um número complexo.

## <a name="return-value"></a>Valor de retorno

A raiz quadrada de *z*. O resultado é mostrado no meio plano direito.

|Entrada|Exceção SEH|**_matherr** Exception|
|-----------|-------------------|--------------------------|
|± QNAN, IND|nenhum|_DOMAIN|
|- ∞|nenhum|_DOMAIN|

## <a name="remarks"></a>Comentários

Como C++ o permite sobrecarga, você pode chamar sobrecargas de **csqrt** que usam e retornam valores **_Fcomplex** e **_Lcomplex** . Em um programa C, **csqrt** sempre pega e retorna um valor **_Dcomplex** .
