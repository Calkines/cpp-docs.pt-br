# <a name="cproj-cprojf-cprojl"></a>cproj, cprojf, cprojl

Recupera a projeção de um número complexo na esfera de Reimann.

## <a name="syntax"></a>Sintaxe

```C
_Dcomplex cproj(
   _Dcomplex z
);
_Fcomplex cproj(
   _Fcomplex z
);  // C++ only
_Lcomplex cproj(
   _Lcomplex z
);  // C++ only
_Fcomplex cprojf(
   _Fcomplex z
);
_Lcomplex cprojl(
   _Lcomplex z
);
```

### <a name="parameters"></a>Parâmetros

*z*<br/>
Um número complexo.

## <a name="return-value"></a>Valor de retorno

A projeção de *z* na esfera de Reimann.

## <a name="remarks"></a>Comentários

Como C++ o permite sobrecarga, você pode chamar sobrecargas de **cproj** que usam e retornam valores **_Fcomplex** e **_Lcomplex** . Em um programa C, **cproj** sempre pega e retorna um valor **_Dcomplex** .
