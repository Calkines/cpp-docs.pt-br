# <a name="_iob"></a>_iob

A matriz de estruturas de controle stdio.

## <a name="syntax"></a>Sintaxe

```
FILE _iob[_IOB_ENTRIES];
```

## <a name="remarks"></a>Comentários

`IOB_ENTRIES` é definido como 20 no stdio.h.
