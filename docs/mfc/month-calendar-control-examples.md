# <a name="month-calendar-control-examples"></a>Exemplos de controle de calendário mensal

O [CMNCTRL1](../overview/visual-cpp-samples.md) aplicativo de exemplo demonstra os vários atributos do `CMonthCalCtrl` classe. O controle, encontrado em uma guia separada na amostra, demonstra a funcionalidade básica e permite que o usuário modificar dinamicamente a determinados atributos.
