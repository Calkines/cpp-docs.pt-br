# <a name="using-cricheditctrl"></a>Usando CRichEditCtrl

Um controle rich edit é uma janela na qual o usuário pode inserir e editar o texto. O texto pode ser atribuído caracteres e formatação de parágrafo e pode incluir objetos OLE incorporados. O controle rich edit é representado no MFC com o [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md) classe.
