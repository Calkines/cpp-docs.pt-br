# <a name="adding-an-mfc-message-handler"></a>Adicionando um manipulador de mensagens do MFC

Você pode [use a janela Propriedades para adicionar um manipulador de mensagens](../../mfc/reference/mapping-messages-to-functions.md) (uma função de membro que lida com mensagens do Windows) para uma classe e mapa de mensagens de Windows para o manipulador de mensagens. Você também pode adicionar [um manipulador de eventos para qualquer controle de caixa de diálogo](../../windows/adding-event-handlers-for-dialog-box-controls.md).

Usando a janela de propriedades para definir funções de manipulação de mensagens e eventos, você pode atualizar automaticamente a tabela de expedição de mensagens (ou mapa de mensagens) e seu arquivo de cabeçalho de classe.

> [!NOTE]
>  Você pode adicionar um manipulador de mensagens para uma classe ATL usando a janela de propriedades; No entanto, alguns resultados podem variar. Para obter mais informações, consulte o tópico ATL [adicionando um manipulador de mensagens do ATL](../../atl/adding-an-atl-message-handler.md).
