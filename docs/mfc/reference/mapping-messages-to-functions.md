# <a name="mapping-messages-to-functions"></a>Mapeando mensagens para funções

O [Assistente de classe](mfc-class-wizard.md) permite que você vincule manipuladores de mensagens (funções de membro de classes de interface de usuário do MFC) às mensagens geradas pelos recursos do seu aplicativo. Eles usam [mapas de mensagens do MFC](../../mfc/messages-and-commands-in-the-framework.md) para criar a associação.

Quando você usa Modo de Exibição de Classe para criar uma nova classe derivada de uma das classes de estrutura, ela automaticamente coloca uma classe completa e funcional nos arquivos de cabeçalho (. h) e de implementação (. cpp) que você especificar.

> [!NOTE]
>  Para adicionar uma nova classe que não manipula mensagens, crie a classe diretamente no editor de texto.

### <a name="to-define-or-remove-a-message-handler-using-the-class-wizard"></a>Para definir ou remover um manipulador de mensagens usando o assistente de classe

1. Em **modo de exibição de classe**, clique com o botão direito do mouse na classe.

1. No menu de contexto, escolha [Assistente de classe](mfc-class-wizard.md).
