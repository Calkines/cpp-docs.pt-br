# <a name="mfc-classes"></a>Classes MFC

> [!TIP]
>  Para obter informações sobre [CStringT](../../atl-mfc-shared/reference/cstringt-class.md) e outras classes que são comuns à programação do MFC e da ATL, consulte [classes compartilhadas por MFC e ATL](../../atl-mfc-shared/reference/classes-shared-by-mfc-and-atl.md).

> [!IMPORTANT]
>  As classes do MFC e seus membros não podem ser usados em aplicativos que são executados no Windows Runtime.
