# <a name="using-the-dialog-editor-to-add-controls"></a>Usando o editor da caixa de diálogos para adicionar controles

Quando você cria seu recurso de modelo de caixa de diálogo com o [editor de caixa de diálogo](../windows/dialog-editor.md), arraste os controles de uma paleta de controles e soltá-los na caixa de diálogo. Isso adiciona as especificações para esse tipo de controle para o recurso de modelo de caixa de diálogo. Quando você constrói um objeto de caixa de diálogo e chamar seus `Create` ou `DoModal` função de membro, o framework cria um controle do Windows e o coloca na janela de diálogo na tela.

Você pode em vez disso [criar controles manualmente](../mfc/adding-controls-by-hand.md) se você quiser. Isso é mais trabalhoso.
