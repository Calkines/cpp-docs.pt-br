# <a name="using-frame-windows"></a>Usando janelas com moldura

Estrutura MFC cria janelas com moldura de documento — e seus documentos e exibições — como parte de sua implementação do novo e Open comandos no menu arquivo. Porque a estrutura faz grande parte do trabalho da janela do quadro para você, você pode reproduzir apenas uma pequena função na criação, uso e destruindo nessas janelas. Você pode, no entanto, explicitamente criar suas próprias janelas de quadro e janelas filho para fins especiais.
