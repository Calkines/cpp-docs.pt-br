# <a name="tn051-using-ctl3d-now-and-in-the-future"></a>TN051: Usando CTL3D agora e no futuro

Essa observação técnica, que é discutido anteriormente CTL3D e MFC, agora é obsoleta. O efeito em 3D para controles automaticamente é implementado pelo MFC.
