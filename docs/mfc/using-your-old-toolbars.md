# <a name="using-your-old-toolbars"></a>Usando as barras de ferramentas anteriores

Se você usou versões anteriores do Visual C++ para criar barras de ferramentas personalizadas, a nova implementação da classe [CToolBar](../mfc/reference/ctoolbar-class.md) pode causar problemas. Para que você não precisa abandonar suas barras de ferramentas para usar a nova funcionalidade, ainda há suporte para a antiga implementação.

O exemplo DOCKTOOL não usa as barras de ferramentas de estilo antigo, somente as barras de ferramentas de novo estilo.

Você não pode editar as barras de ferramentas de estilo antigo com o editor de recurso da barra de ferramentas.
