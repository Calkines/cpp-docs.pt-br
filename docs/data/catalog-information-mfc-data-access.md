# <a name="catalog-information--mfc-data-access"></a>Informações de catálogo (Acesso a dados MFC)

Informações sobre as tabelas em uma fonte de dados podem incluir nomes de tabelas com suas respectivas colunas, privilégios de tabela, nomes de chaves primária e externa, informações sobre consultas predefinidas ou procedimentos armazenados, informações sobre índices nas tabelas e estatísticas sobre tabelas.

Para obter mais informações, consulte [fonte de dados: Determinando o esquema da fonte de dados (ODBC)](../data/odbc/data-source-determining-the-schema-of-the-data-source-odbc.md).
