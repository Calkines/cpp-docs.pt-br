# <a name="no_search_namespace-import-attribute"></a>atributo de importação de no_search_namespace

**C++Determinados**

Tem a mesma funcionalidade que o atributo [no_namespace](../preprocessor/no-namespace.md) , mas é usado em bibliotecas de tipos em que você `#import` usa a diretiva com o atributo [auto_search](../preprocessor/auto-search.md) .

## <a name="syntax"></a>Sintaxe

> **#import** *biblioteca de tipos* **no_search_namespace**

**FINAL C++ específico**
