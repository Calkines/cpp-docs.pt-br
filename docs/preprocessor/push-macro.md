# <a name="push_macro-pragma"></a>Pragma push_macro

Salva o valor da macro de *nome de macro* na parte superior da pilha para esta macro.

## <a name="syntax"></a>Sintaxe

> **#pragma push_macro ("** _nome da macro_ **")**

## <a name="remarks"></a>Comentários

Você pode recuperar o valor de *macro-Name* com `pop_macro`.

Consulte [pop_macro](../preprocessor/pop-macro.md) para obter um exemplo.
