# <a name="omp-pragma"></a>Pragma omp

Usa uma ou mais políticas de OpenMP, junto com quaisquer cláusulas de políticas opcionais.

## <a name="syntax"></a>Sintaxe

> **#pragma OMP** *diretiva* do

## <a name="remarks"></a>Comentários

Para obter mais informações, consulte [diretivas de OpenMP](../parallel/openmp/reference/openmp-directives.md).
