# <a name="platformenum-class"></a>Classe Platform::Enum

Uma classe de valor que representa um conjunto de constantes nomeadas.

## <a name="syntax"></a>Sintaxe

```cpp
public class Enum
```

### <a name="members"></a>Membros

A classe Enum herda os métodos Equals(), GetHashCode() e ToString() da [Platform::Object Class](../cppcx/platform-object-class.md).

### <a name="remarks"></a>Comentários

Use a palavra-chave [classe pública de enum](../extensions/enum-class-cpp-component-extensions.md) para criar enumerações. Não use o tipo Platform::Enum explicitamente. Para obter mais informações, consulte [Enums](../cppcx/enums-c-cx.md).

### <a name="requirements"></a>Requisitos

**Cliente com suporte mínimo:** Windows 8

**Servidor com suporte mínimo:** Windows Server 2012

**Namespace:** Plataforma

**Metadados:** platform.winmd
