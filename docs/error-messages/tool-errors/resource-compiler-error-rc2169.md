# <a name="resource-compiler-error-rc2169"></a>Erro RC2169 (compilador de recurso)

nome de arquivo de recurso do arquivo não está no formato 2,03

O recurso especificado usado em um formato anterior à versão 2.03. O arquivo de recurso deve ser convertido ou recriado usando o formato de versão 3,00 ou posterior.
