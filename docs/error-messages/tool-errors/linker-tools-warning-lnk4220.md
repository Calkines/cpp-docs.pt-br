# <a name="linker-tools-warning-lnk4220"></a>Aviso LNK4220 (Ferramentas de Vinculador)

inválido 'opção de vinculador' valor 'valor'; padrão assumida

Um valor fora do intervalo foi especificado com o [/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md) opção. O valor padrão para **/TLBID** é 1.
