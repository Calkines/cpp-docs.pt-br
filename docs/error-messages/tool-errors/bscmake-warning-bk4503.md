# <a name="bscmake-warning-bk4503"></a>Aviso BK4503 (BSCMAKE)

Erro secundário no nome de arquivo do arquivo SBR ignorado

O arquivo .sbr continha um erro que não interrompia a compilação, mas o arquivo .bsc resultante pode estar incorreto. Para corrigir esse problema, recompile para criar um novo .sbr.
