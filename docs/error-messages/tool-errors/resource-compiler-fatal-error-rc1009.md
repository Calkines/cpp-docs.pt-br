# <a name="resource-compiler-fatal-error-rc1009"></a>Erro fatal RC1009 (compilador de recurso)

limite do compilador: macros com aninhamento muito profundo 'macro'

O arquivo excede o limite do compilador de recurso para o aninhamento de macro. Revise o arquivo de origem especificado para diminuir a profundidade de aninhamento de suas macros.
