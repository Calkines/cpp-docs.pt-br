# <a name="linker-tools-error-lnk1282"></a>Erro das Ferramentas de Vinculador LNK1282

não é possível /REBASE arquivo; ele foi assinado

Você tentou alterar o endereço base de um assembly assinado com a opção /REBASE [editbin](../../build/reference/editbin-reference.md). Para fazer isso, primeiro altere o endereço básico e, em seguida, assinar o assembly.
