# <a name="linker-tools-error-lnk1143"></a>Erro das Ferramentas de Vinculador LNK1143

arquivo inválido ou corrompido: nenhum símbolo para o número de seção COMDAT

Esse erro pode ser causado se o arquivo de objeto está corrompido.

Recompile o arquivo.
