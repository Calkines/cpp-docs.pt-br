# <a name="expression-evaluator-error-cxx0058"></a>Erro CXX0058 (avaliador de expressão)

operador sobrecarregado não encontrado

Um tipo de classe foi especificado como o operando esquerdo em uma expressão, mas um operador sobrecarregado não foi definido para a classe.

Esse erro é idêntico ao CAN0058.
