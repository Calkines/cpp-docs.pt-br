# <a name="linker-tools-warning-lnk4096"></a>Aviso LNK4096 (Ferramentas de Vinculador)

/ Valor BASE "number" é inválido para o Windows 95 e Windows 98. imagem não pode ser executada

O endereço base especificado é inválido. Arquivos executáveis do Windows 95 e Windows 98 devem ter um endereço base maior 0x400000. Para obter mais informações sobre endereços base, consulte o [/base](../../build/reference/base-base-address.md) a opção de vinculador.
