# <a name="expression-evaluator-error-cxx0007"></a>Erro CXX0007 (avaliador de expressão)

parêntese direito ausente

Parênteses faltando foram encontrados na expressão. Digite novamente a expressão entre parênteses equilibradas.

Esse erro é idêntico ao CAN0007.
