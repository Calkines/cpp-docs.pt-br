# <a name="project-build-error-prj0017"></a>Erro de Build PRJ0017 no Projeto

Diretório de trabalho atual é inválido.

O caminho para o diretório de trabalho atual é maior que _MAXPATH. Para resolver esse erro, não crie seu projeto em um nível profundo.
