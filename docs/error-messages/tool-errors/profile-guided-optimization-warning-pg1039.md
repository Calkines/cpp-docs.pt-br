# <a name="profile-guided-optimization-warning-pg1039"></a>Aviso PG1039 (otimização guiada por perfil)

Falha na espera para tempo de espera para varredura.

O aplicativo não permitiu [pgosweep](../../build/pgosweep.md) para ser executado.  Se possível, execute pgosweep quando o aplicativo está em um estado ocioso.
