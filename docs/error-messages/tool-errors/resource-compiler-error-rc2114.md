# <a name="resource-compiler-error-rc2114"></a>Erro RC2114 (compilador de recurso)

nome de classe de controle esperado

O `class` campo de uma instrução de controle em de **caixa de diálogo** instrução deve ser um dos seguintes tipos: BOTÃO, COMBOBOX, editar, LISTBOX, barra de ROLAGEM, estático, ou definido pelo usuário. Verifique se que a classe está escrita corretamente.
