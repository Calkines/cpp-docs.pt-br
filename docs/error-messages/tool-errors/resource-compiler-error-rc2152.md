# <a name="resource-compiler-error-rc2152"></a>Erro RC2152 (compilador de recurso)

caractere de controle inválido

Um caractere de controle na **ACELERADORES** instrução é inválida. Um caractere de controle válido consiste em uma letra (somente) seguindo um acento circunflexo (**^**).
