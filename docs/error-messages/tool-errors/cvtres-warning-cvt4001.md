# <a name="cvtres-warning-cvt4001"></a>Aviso CVTRES CVT4001

> tipo de máquina não estiver especificado; supõe-se *tipo*

CVTRES não encontrou uma especificação de máquina. Pressupõe-se o tipo de determinado computador. Se o padrão é CVTRES incorreto, execute novamente usando a opção /MACHINE.
