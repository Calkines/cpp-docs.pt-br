# <a name="expression-evaluator-error-cxx0054"></a>Erro CXX0054 (avaliador de expressão)

Registre-se na expressão de ponto de interrupção inválido

Um registro não pode ser usado em uma expressão de ponto de interrupção.

Esse erro é idêntico ao CAN0054.
