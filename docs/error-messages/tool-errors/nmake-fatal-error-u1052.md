# <a name="nmake-fatal-error-u1052"></a>Erro fatal U1052 (NMAKE)

> arquivo '*filename*' não encontrado

NMAKE não foi possível encontrar o arquivo especificado com um dos seguintes:

- **/F** opção

- **! INCLUIR** diretiva de pré-processamento

- Sinal de arroba (**\@**) especificador para um arquivo de resposta

Verifique se o arquivo existe e o nome do arquivo está escrito corretamente.
