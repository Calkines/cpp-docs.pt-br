# <a name="linker-tools-errors-and-warnings-lnkxxxx"></a>Erros e avisos das ferramentas do Vinculador (LNKxxxx)

As ferramentas de vinculador LINK, LIB, DUMPBIN e EDITBIN compartilham um executável comuns que gera esses erros e avisos. As ferramentas geram avisos do formulário LNK*xxxx*, onde *xxxx* é um número de quatro dígitos.
