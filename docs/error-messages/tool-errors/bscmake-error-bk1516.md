# <a name="bscmake-error-bk1516"></a>Erro BK1516 (BSCMAKE)

bscfile corrompido; não é possível atualizar de forma incremental

O arquivo. bsc foi corrompido, possivelmente devido a uma falha do sistema durante a compilação. Exclua o arquivo. bsc, recompile todos os arquivos. SBR e recompile o arquivo. bsc.
