# <a name="cvtres-fatal-error-cvt1105"></a>Erro fatal CVT1105 (CVTRES)

não é possível buscar no arquivo

CVTRES não foi possível ir para um local no arquivo.

Esse erro pode ser causado por corrupção de arquivo.
