# <a name="compiler-error-c2173"></a>Erro do compilador C2173

'function': parâmetro real não for um ponteiro: parâmetro Número1, Número2 da lista de parâmetro

Parâmetro `number1` passado para a lista de parâmetros `number2` não for um ponteiro. A função espera um ponteiro.
