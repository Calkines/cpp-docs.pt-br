# <a name="fatal-error-c1093"></a>Erro fatal C1093

> Chamada de API '*nome da função*' falhou '*local da chamada*': '*texto de tempo de execução*'

Falha em uma chamada para uma função .NET. O *texto da cadeia de tempo de execução* pode ou não ser fornecido pelo tempo de execução com.

Para obter mais informações sobre mensagens de erro do sistema, consulte o arquivo de sistema Winerror. h e [FormatMessage](/windows/win32/api/winbase/nf-winbase-formatmessage).
