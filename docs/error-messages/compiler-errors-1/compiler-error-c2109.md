# <a name="compiler-error-c2109"></a>Erro do compilador C2109

subscrito requer tipo ponteiro ou matriz

O subscrito foi usado em uma variável que não era uma matriz.

O exemplo a seguir gera C2109:

```
// C2109.cpp
int main() {
   int a, b[10] = {0};
   a[0] = 1;   // C2109
   b[0] = 1;   // OK
}
```
