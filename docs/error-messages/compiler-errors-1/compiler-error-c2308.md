# <a name="compiler-error-c2308"></a>Erro do compilador C2308

concatenação de cadeias de caracteres incompatíveis

Ambas as cadeias de caracteres largos e não todo o foram especificadas para concatenação. Você não é possível concatenar uma cadeia de caracteres largos e a cadeia de caracteres não-wide.
