# <a name="fatal-error-c1350"></a>Erro fatal C1350

Erro ao carregar a dll 'dll': dll não encontrada

A DLL que oferece suporte a tentativa de operação não foi encontrada.

Isso indica um problema com sua instalação e você deve reinstalar a DLL a partir do CD do produto.
