# <a name="fatal-error-c1098"></a>Erro fatal C1098

Incompatibilidade de versão com mecanismo editar e continuar

A versão do depurador que você está usando não coincide com o compilador usado para criar o executável. Se a recompilação não corrigir o problema, você talvez precise reinstalar o Visual C++ para garantir que você tem as versões apropriadas do depurador e do compilador.
