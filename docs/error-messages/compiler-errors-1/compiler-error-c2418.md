# <a name="compiler-error-c2418"></a>Erro do Compilador C2418

não é possível excluir o arquivo de navegador: nome de arquivo

O compilador não pôde excluir o arquivo de navegador.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Para corrigir verificando as possíveis causas a seguir

1. Arquivo é somente leitura.

1. Arquivo está em uso por outro processo.
