# <a name="fatal-error-c1057"></a>Erro fatal C1057

fim de arquivo inesperado na expansão de macro

O compilador atingiu o fim do arquivo de origem durante a obtenção de argumentos de invocação de macro, provavelmente devido a um parêntese direito ausente na invocação de macro.
