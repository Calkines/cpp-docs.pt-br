# <a name="compiler-error-c2412"></a>Erro do compilador C2412

'Rótulo': rótulo diferencia maiusculas de minúsculas redefinido

O rótulo é definido mais de uma vez na função atual. Altere a ortografia do rótulo e suas referências.
