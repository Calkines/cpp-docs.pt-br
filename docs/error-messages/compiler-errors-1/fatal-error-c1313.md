# <a name="fatal-error-c1313"></a>Erro fatal C1313

limite do compilador: blocos de tipo não podem ser aninhados mais profunda do que os níveis de número

Exceção tratamento (ou manipulação de exceção estruturada) blocos estavam aninhados muito profundamente.  Simplifique seu código.
