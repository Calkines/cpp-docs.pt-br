# <a name="fatal-error-c1023"></a>Erro fatal C1023

'file': erro inesperado com pch, tente recompilá-lo

C1023 pode ser causado por um de vários problemas, a solução para o qual é uma recompilação do arquivo de cabeçalho pré-compilado.

C1023 pode ser causado se você tentar usar um arquivo de cabeçalho pré-compilado em um computador diferente no qual ele foi criado.
