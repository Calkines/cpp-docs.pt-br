# <a name="compiler-error-c2224"></a>Erro do compilador C2224

esquerda de '.identifier' deve ter o tipo de struct/union

O operando à esquerda do período (.) não é uma classe, estrutura ou união.

Esse erro pode ser causado por um operando à esquerda que é uma variável indefinida (portanto, digite `int`).
