# <a name="compiler-error-c3275"></a>Erro do compilador C3275

'membro de enumeração ': não é possível usar este símbolo sem qualificador

Ao usar o código gerenciado e quando duas ou mais enumerações contêm um identificador com o mesmo nome, você deve qualificar explicitamente referências para o identificador.

C3275 só está acessível usando a opção de compilador obsoletos **/CLR: oldSyntax**.
