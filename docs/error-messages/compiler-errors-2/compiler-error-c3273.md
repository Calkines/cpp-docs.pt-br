# <a name="compiler-error-c3273"></a>Erro do compilador C3273

Finally não pode ser usado em um bloco de exceção no código não gerenciado.

O exemplo a seguir gera C3273:

```
// C3273.cpp
// compile with: /GX
int main()
{
   try
   {
   }
   catch (int)
   {
   }
   __finally   // C3273, remove __finally clause
   {
   }
}
```
