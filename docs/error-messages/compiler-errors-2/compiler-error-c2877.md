# <a name="compiler-error-c2877"></a>Erro do compilador C2877

'symbol' não é acessível de 'class'

Todos os membros derivados de uma classe base devem ser acessíveis na classe derivada.

O exemplo a seguir gera C2877:

```
// C2877.cpp
// compile with: /c
class A {
private:
   int a;
};

class B : public A {
   using A::a;   // C2877
};
```
