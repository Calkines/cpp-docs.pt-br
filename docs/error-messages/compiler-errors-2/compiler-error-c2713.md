# <a name="compiler-error-c2713"></a>Erro do compilador C2713

apenas uma forma de tratamento de exceção permitida por função

Não é possível usar o tratamento de exceções estruturado (`__try`/`__except`) e o tratamento de exceção do C++ (`try`/`catch`) na mesma função.
