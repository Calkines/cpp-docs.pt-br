# <a name="compiler-error-c3721"></a>Erro do compilador C3721

'assinatura': assinatura incompatível para evento

Um evento foi declarado incorretamente. Para obter mais informações, consulte [Event](../../cpp/event.md).

C3721 só está acessível usando a opção de compilador obsoletos **/CLR: oldSyntax**.
