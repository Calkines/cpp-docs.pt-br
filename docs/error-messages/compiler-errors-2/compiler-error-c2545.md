# <a name="compiler-error-c2545"></a>Erro do compilador C2545

'operator': não é possível encontrar operador sobrecarregado

O operador não pode ser usado com operandos fornecidos. Você deve fornecer um operador sobrecarregado com os operandos necessários.

Esse erro pode ser causado se operandos tiverem o tipo incorreto.

Esse erro pode ser corrigido se você definir um operador de conversão ou um construtor que aceita um único parâmetro.
