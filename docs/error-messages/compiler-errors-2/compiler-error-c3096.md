# <a name="compiler-error-c3096"></a>Erro do compilador C3096

'attribute': atributo é permitido em membros de classes de atributo apenas dados

Um atributo foi aplicado incorretamente.

Para obter mais informações, consulte [atributos definidos pelo usuário](../../extensions/user-defined-attributes-cpp-component-extensions.md).
