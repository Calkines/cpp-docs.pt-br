# <a name="compiler-error-c2974"></a>Erro do compilador C2974

tipo inválido argumento 'número', tipo esperado

O argumento de modelo ou genérico não coincide com a declaração genérico ou modelo. Um tipo deve aparecer dentro de colchetes angulares. Verifique a definição de modelo ou genérico para localizar os tipos corretos.

O exemplo a seguir gera C2974:

```
// C2974.cpp
// C2974 expected
template <class T>
struct TC {};

template <typename T>
void tf(T){}

int main() {
   // Delete the following 2 lines to resolve
   TC<1>* tc;
   tf<"abc">("abc");

   TC<int>* tc;
   tf<const char *>("abc");
}
```

C2974 também podem ocorrer ao usar genéricos:

```
// C2974b.cpp
// compile with: /clr
// C2974 expected
using namespace System;
generic <class T>
ref struct GCtype {};

generic <typename T>
void gf(T){}

int main() {
   // Delete the following 2 lines to resolve
   GCtype<"a">^ gc;
   gf<"a">("abc");

   // OK
   GCtype<int>^ gc;
   gf<String ^>("abc");
}
```
