# <a name="compiler-warning-level-1-c4029"></a>Compilador aviso (nível 1) C4029

lista de parâmetros formais declarada diferente da definição

Tipos de parâmetro formal na declaração de função não concordam com aqueles na definição de função. O compilador usa a lista de parâmetros da definição.
