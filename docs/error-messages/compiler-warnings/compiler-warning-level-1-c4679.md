# <a name="compiler-warning-level-1-c4679"></a>Compilador aviso (nível 1) C4679

> '*membro*': não foi possível importar membro

O compilador encontrou uma construção que não tem suporte, que não pode ser importada dos metadados.

Não tente usar a construção.
