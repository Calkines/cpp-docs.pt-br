# <a name="compiler-warning-level-1-c4045"></a>Compilador aviso (nível 1) C4045

'array': estouro nos limites da matriz

A matriz tem muitos inicializadores. Inicializadores extras serão ignorados.

Certifique-se de que inicializadores e elementos de matriz correspondam em tamanho e quantidade.
