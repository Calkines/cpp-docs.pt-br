# <a name="compiler-warning-level-1-c4026"></a>Compilador aviso (nível 1) C4026

função declarada com lista de parâmetros formais

A declaração de função possui parâmetros formais, mas não a definição da função. As chamadas subsequentes para essa função pressupõem que a função não usa nenhum parâmetro.
