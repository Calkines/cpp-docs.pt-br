# <a name="compiler-warning-level-1-c4088"></a>Compilador aviso (nível 1) C4088

'function': incompatibilidade de ponteiro em parâmetro real 'número', 'number' do parâmetro formal

Os parâmetros formais e reais correspondentes têm diferentes níveis de indireção. O parâmetro real será passado sem alteração. A função chamada interpreta seu valor como um ponteiro.
