# <a name="compiler-warning-level-3-c4619"></a>Compilador aviso (nível 3) C4619

\#Aviso de Pragma: há um número de aviso 'number'

Foi feita uma tentativa para desabilitar um aviso de que não existe.

Esse aviso é desativado por padrão. Ver [compilador avisos que são desativado por padrão](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para obter mais informações.

O exemplo a seguir gera C4619:

```
// C4619.cpp
// compile with: /W3 /c
#pragma warning(default : 4619)
#pragma warning(disable : 4354)   // C4619, C4354 does not exist
```
