# <a name="compiler-warning-level-1-c4143"></a>Compilador aviso (nível 1) C4143

pragma 'same_seg' não tem suportada; Use alocação based

O **#pragma same_seg** não é mais suportada. Use o [based](../../cpp/based-pointers-cpp.md) palavra-chave em vez disso.
