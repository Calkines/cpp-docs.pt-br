# <a name="compiler-warning-level-1-c4811"></a>Compilador aviso (nível 1) C4811

valor de pragma conform (forScope, show) = = valor

Esse aviso é emitido quando você usa o **mostram** opção do [estão em conformidade com](../../preprocessor/conform.md) pragma. *valor* é o valor atual de conformidade.
