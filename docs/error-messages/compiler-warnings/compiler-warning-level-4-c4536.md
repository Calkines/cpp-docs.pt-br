# <a name="compiler-warning-level-4-c4536"></a>Compilador aviso (nível 4) C4536

'type name': nome do tipo excede o limite de metadados de caracteres 'limit'

Um nome de tipo seria truncado em metadados se fosse um tipo gerenciado. Ver [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md) para obter mais informações.

Esse aviso é desativado por padrão. Ver [compilador avisos que são desativado por padrão](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para obter mais informações.
