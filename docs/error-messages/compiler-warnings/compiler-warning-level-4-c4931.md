# <a name="compiler-warning-level-4-c4931"></a>Compilador aviso (nível 4) C4931

> estamos supondo que a biblioteca de tipos foi criada para o *número*-ponteiros de bit

Informações explícitas não foi fornecidas com o **ptrsize** atributo da [#import](../../preprocessor/hash-import-directive-cpp.md) diretiva; o compilador concluiu que o tamanho do ponteiro da biblioteca de tipos é *número*.

Esse aviso é desativado por padrão. Ver [compilador avisos que são desativado por padrão](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para obter mais informações.
