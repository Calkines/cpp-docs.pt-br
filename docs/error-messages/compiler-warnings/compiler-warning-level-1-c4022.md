# <a name="compiler-warning-level-1-c4022"></a>Compilador aviso (nível 1) C4022

'function': incompatibilidade de ponteiro para o parâmetro real 'number'

O tipo de ponteiro do parâmetro real é diferente do tipo de ponteiro de parâmetro formal correspondente. O parâmetro real será passado sem alteração.
