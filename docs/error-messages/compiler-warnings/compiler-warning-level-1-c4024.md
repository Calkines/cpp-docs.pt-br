# <a name="compiler-warning-level-1-c4024"></a>Compilador aviso (nível 1) C4024

'function': tipos diferentes para o parâmetro formal e real 'number'

Parâmetros formais e reais correspondentes têm tipos diferentes. O compilador passa o parâmetro real sem alteração. A função de receptor converte o tipo de parâmetro para o tipo esperado.
