# <a name="compiler-warning-level-4-c4505"></a>Compilador aviso (nível 4) C4505

'function': função local não referenciada foi removida

A função fornecida é local e não é referenciado no corpo do módulo; Portanto, a função é código inativo.

O compilador não gerar código para esta função inativo.
