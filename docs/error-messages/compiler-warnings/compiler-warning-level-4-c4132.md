# <a name="compiler-warning-level-4-c4132"></a>Compilador aviso (nível 4) C4132

'object': objeto const deve ser inicializado

A constante não foi inicializada. O valor de um símbolo com a **const** atributo não pode ser alterado após sua definição, ele deve ser fornecido um valor de inicialização.
