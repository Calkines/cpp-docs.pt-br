# <a name="compiler-warning-level-1-c4158"></a>Compilador aviso (nível 1) C4158

> Supondo que #pragma pointers_to_members (full_generality, herança)

Um **#pragma pointers_to_members (** *único* &#124; *vários* &#124; *virtual* **)** foi emitido sem um associada **pointers_to_members (full_generality) #pragma**.
