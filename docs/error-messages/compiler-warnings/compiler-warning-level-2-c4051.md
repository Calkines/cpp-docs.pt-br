# <a name="compiler-warning-level-2-c4051"></a>Compilador aviso (nível 2) C4051

conversão de tipo; possível perda de dados

Uma expressão contém dois itens de dados com diferentes tipos de base. Conversão de um tipo faz com que o item de dados sejam truncados.

Esse aviso pode ser corrigido se você converter os itens de dados para o tipo apropriado.
