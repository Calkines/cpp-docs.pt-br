# <a name="compiler-warning-level-1-c4122"></a>Compilador aviso (nível 1) C4122

'function': alloc_text aplicável apenas a funções com vínculo a C

O **alloc_text** pragma se aplica somente a funções declaradas com **extern do c**. Ele não pode ser usado com funções externas de C++.

O pragma é ignorado.
