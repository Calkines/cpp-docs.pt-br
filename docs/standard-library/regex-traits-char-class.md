# <a name="regextraitsltchargt-class"></a>Classe regex_traits&lt;char&gt;

Especialização `regex_traits` de para **Char**.

## <a name="syntax"></a>Sintaxe

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>Comentários

A classe é uma especialização explícita da classe template [regex_traits](../standard-library/regex-traits-class.md) para elementos do tipo **Char** (para que ele possa tirar proveito das funções de biblioteca que manipulam objetos desse tipo).

## <a name="requirements"></a>Requisitos

**Cabeçalho:** \<regex>

**Namespace:** std
