# <a name="ltciso646gt"></a>&lt;ciso646&gt;

Inclui o cabeçalho \<iso646.h> da biblioteca C Padrão e adiciona os nomes associados ao namespace `std`.

> [!NOTE]
> O uso desse cabeçalho é desnecessário em C++.

## <a name="requirements"></a>Requisitos

**Cabeçalho:** \<ciso646 >

**Namespace:** std

## <a name="remarks"></a>Comentários

A inclusão desse cabeçalho garante que os nomes declarados usando vinculação externa no cabeçalho da biblioteca C Padrão sejam declarados no namespace `std`.
