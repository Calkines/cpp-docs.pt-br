## <a name="swap"></a>  swap

Troca os elementos dos dois deques.

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>Parâmetros

*left*<br/>
Um objeto do tipo `deque`.

*right*<br/>
Um objeto do tipo `deque`.
