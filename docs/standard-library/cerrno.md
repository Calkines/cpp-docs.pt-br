# <a name="ltcerrnogt"></a>&lt;cerrno&gt;

Inclui o cabeçalho da biblioteca C padrão \<errno. h > e adiciona os nomes associados ao `std` namespace. Inclusão desse cabeçalho garante que os nomes declarados usando vinculação externa no cabeçalho da biblioteca C padrão sejam declarados no `std` namespace.

## <a name="syntax"></a>Sintaxe

```cpp
#include <cerrno>
```
