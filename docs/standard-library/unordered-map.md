# <a name="ltunorderedmapgt"></a>&lt;unordered_map&gt;

## <a name="syntax"></a>Sintaxe

```cpp
#include <unordered_map>
```

> [!NOTE]
> A \<biblioteca de > unordered_map também usa `#include <initializer_list>` a instrução.
