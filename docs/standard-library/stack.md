# <a name="ltstackgt"></a>&lt;stack&gt;

Define a pilha de classe de modelo e os dois modelos de suporte.

## <a name="requirements"></a>Requisitos

**Cabeçalho:** \<stack>

**Namespace:** std

> [!NOTE]
> O \<pilha > biblioteca também usa o `#include <initializer_list>` instrução.
