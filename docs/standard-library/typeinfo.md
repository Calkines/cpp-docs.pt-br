# <a name="lttypeinfogt"></a>&lt;typeinfo&gt;

Inclua o cabeçalho padrão \<typeinfo > para definir vários tipos associados com o operador de identificação de tipo [operador typeid](../cpp/typeid-operator.md), que traz informações sobre tipos estáticos e dinâmicos.

## <a name="syntax"></a>Sintaxe

```cpp
#include <typeinfo>

```

- [Exceção bad_cast](../cpp/bad-cast-exception.md)

- [Exceção bad_typeid](../cpp/bad-typeid-exception.md)

- [Classe type_info](../cpp/type-info-class.md)
