# <a name="instruction-set-for-inline-assembly"></a>Conjunto de instruções para assembly embutido

**Seção específica da Microsoft**

O Microsoft C++ compilador é compatível com todos os opcodes por meio do Pentium 4 e AMD Athlon. Instruções adicionais compatíveis com o processador de destino podem ser criadas com o [pseudoinstrução Emit](../../assembler/inline/emit-pseudoinstruction.md).
