# <a name="masm-expressions-in-inline-assembly"></a>Expressões MASM em assembly embutido

**Seção específica da Microsoft**

Código de assembly embutido pode usar qualquer expressão (MASM), que é qualquer combinação de operadores e operandos que é avaliada como um único valor ou o endereço.
