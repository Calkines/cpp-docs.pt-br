# <a name="while-masm"></a>WHILE (MASM)

Repete o assembly do bloco *instruções* enquanto *expressão* permanece verdadeira.

## <a name="syntax"></a>Sintaxe

> ENQUANTO *expressão*<br/>
> *Instruções*<br/>
> ENDM
