# <a name="df"></a>DF

Aloca e inicializa opcionalmente 6 bytes de armazenamento para cada `initializer`. `DF` é um sinônimo de [FWORD](../../assembler/masm/fword.md).

## <a name="syntax"></a>Sintaxe

> [[nome]] Inicializador de DF [[, inicializador]]...

## <a name="remarks"></a>Comentários

Também pode ser usado como um especificador de tipo em qualquer lugar que um tipo é legal.
