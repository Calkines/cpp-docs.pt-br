# <a name="real4"></a>REAL4

Aloca e inicializa, opcionalmente, um número de ponto flutuante de precisão simples (4 bytes) para cada *inicializador*.

## <a name="syntax"></a>Sintaxe

> inicializador de REAL4 nome [[, inicializador]]...
