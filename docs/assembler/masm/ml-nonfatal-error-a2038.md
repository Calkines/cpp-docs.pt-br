# <a name="ml-nonfatal-error-a2038"></a>Erro não fatal A2038 (ML)

**operando ausente para o operador de macro**

O montador encontrado o final da lista de parâmetros de uma macro imediatamente após o [!](../../assembler/masm/operator-logical-not-masm-run-time.md) ou [ % ](../../assembler/masm/operator-percent.md) operador.
