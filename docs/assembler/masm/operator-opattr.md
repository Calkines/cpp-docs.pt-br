# <a name="operator-opattr"></a>operador OPATTR

Retorna uma palavra definindo o modo e o escopo de *expressão*. O byte inferior é idêntico do byte retornado por [. TIPO](../../assembler/masm/operator-dot-type.md). O byte superior contém informações adicionais.

## <a name="syntax"></a>Sintaxe

> Expressão OPATTR
