# <a name="label-masm"></a>LABEL (MASM)

Cria um novo rótulo atribuindo o valor de contador do local atual e a determinada *tipo* à *nome*.

## <a name="syntax"></a>Sintaxe

> *nome da* rótulo *tipo*

> *nome* rótulo [[NEAR | MUITO | PTR PROC]] [[*tipo*]]
