# <a name="real8"></a>REAL8

Aloca e inicializa, opcionalmente, um número de ponto flutuante de precisão dupla (8 bytes) para cada *inicializador*.

## <a name="syntax"></a>Sintaxe

> inicializador de REAL8 nome [[, inicializador]]...
