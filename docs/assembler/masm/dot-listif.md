# <a name="listif"></a>.LISTIF

Inicia a listagem de instruções em blocos condicionais falsos.

## <a name="syntax"></a>Sintaxe

> .LISTIF

## <a name="remarks"></a>Comentários

Mesmo que [. LFCOND](../../assembler/masm/dot-lfcond.md).
