# <a name="data"></a>@data

O nome do grupo de dados padrão. É avaliada como DGROUP para todos os modelos, exceto **simples**. É avaliada como **simples** sob o **simples** (macro text) do modelo de memória.

## <a name="syntax"></a>Sintaxe

> @data
