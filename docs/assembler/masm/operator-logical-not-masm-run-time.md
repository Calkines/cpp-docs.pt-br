# <a name="operator--masm-run-time"></a>operator ! (Tempo de execução MASM)

Negação lógica. Usado somente em [. IF](../../assembler/masm/dot-if.md), [. Embora](../../assembler/masm/dot-while.md), ou [. REPITA](../../assembler/masm/dot-repeat.md) bloqueia e avaliado em tempo de execução, não no momento do assembly.

## <a name="syntax"></a>Sintaxe

> ! *Expressão*
