# <a name="comment-masm"></a>COMMENT (MASM)

Trata todos *texto* entre ou na mesma linha como delimitadores como um comentário.

## <a name="syntax"></a>Sintaxe

> Delimitador de comentário [[texto]]<br/>
> [[texto]]<br/>
> delimitador de [[texto]] [[texto]]
