# <a name="pushcontext"></a>PUSHCONTEXT

Salva todo ou parte do atual `context`: registro segmento pressupõe, valor fracionário, sinalizadores de listagem e cref ou valores de processador/coprocessador. O `context` pode ser **PRESUME**, `RADIX`, **listando**, **CPU**, ou **todos os**.

## <a name="syntax"></a>Sintaxe

> Contexto PUSHCONTEXT
