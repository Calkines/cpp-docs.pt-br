# <a name="operator-zero"></a>operador ZERO?

Status do sinalizador de zero. Usado somente em [. IF](../../assembler/masm/dot-if.md), [. Embora](../../assembler/masm/dot-while.md), ou [. REPITA](../../assembler/masm/dot-repeat.md) bloqueia e avaliado em tempo de execução, não no momento do assembly.

## <a name="syntax"></a>Sintaxe

> ZERO?
