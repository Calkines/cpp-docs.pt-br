# <a name="ml-nonfatal-error-a2189"></a>Erro não fatal A2189 (ML)

**combinação inválida com alinhamento de segmento**

O alinhamento especificado pelo **ALINHAR** ou `EVEN` diretiva era maior que o alinhamento do segmento atual conforme especificado pelo **segmento** diretiva.
