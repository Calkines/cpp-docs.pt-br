# <a name="operator-gt-masm-run-time"></a>operador &gt;= (tempo de execução MASM)

É maior que ou igual a. Usado somente em [. IF](../../assembler/masm/dot-if.md), [. Embora](../../assembler/masm/dot-while.md), ou [. REPITA](../../assembler/masm/dot-repeat.md) bloqueia e avaliado em tempo de execução, não no momento do assembly.

## <a name="syntax"></a>Sintaxe

> Expression1 > = expression2
