# <a name="fardata"></a>.FARDATA

Quando usado com [. MODELO](../../assembler/masm/dot-model.md), inicia um segmento de dados mais distante para dados inicializados (nome de segmento FAR_DATA ou *nome*).

## <a name="syntax"></a>Sintaxe

> .FARDATA [[name]]
