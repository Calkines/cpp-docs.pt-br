# <a name="operator-sectionrel"></a>operador SECTIONREL

Retorna o deslocamento relativo da seção de expressão em relação a seção que contém o destino no executável final.

## <a name="syntax"></a>Sintaxe

> Expressão SECTIONREL

## <a name="remarks"></a>Comentários

SECTIONREL está disponível somente com a emissão de objeto COFF.
