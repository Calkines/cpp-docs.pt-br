# <a name="listmacroall"></a>.LISTMACROALL

Inicia a listagem de todas as instruções em macros.

## <a name="syntax"></a>Sintaxe

> .LISTMACROALL

## <a name="remarks"></a>Comentários

Mesmo que [. LALL](../../assembler/masm/dot-lall.md).
