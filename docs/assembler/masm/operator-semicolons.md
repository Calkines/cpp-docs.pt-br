# <a name="operator-"></a>Operador ;;

Trata *texto* como um comentário em uma macro que aparece somente na definição da macro. Não mostra a listagem *texto* onde a macro é expandida.

## <a name="syntax"></a>Sintaxe

> ;; *text*
