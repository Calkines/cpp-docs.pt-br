# <a name="ifidn-ifidni"></a>IFIDN, IFIDNI

Conjunto de concessões se os itens de texto são idênticos.

## <a name="syntax"></a>Sintaxe

> **IFIDN**\[**I**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Comentários

Se **IFIDNI** é fornecido, a comparação não diferencia maiusculas de minúsculas. Ver [IF](../../assembler/masm/if-masm.md) para a sintaxe completa.
