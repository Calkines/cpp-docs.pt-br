# <a name="real10"></a>REAL10

Aloca e inicializa, opcionalmente, um número de ponto flutuante de 10 bytes para cada *inicializador*.

## <a name="syntax"></a>Sintaxe

> inicializador de REAL10 nome [[, inicializador]]...
