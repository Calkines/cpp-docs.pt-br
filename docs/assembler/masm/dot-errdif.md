# <a name="perioderrdif-perioderrdifi"></a>&period;ERRDIF, &period;ERRDIFI

Gera um erro se os itens de texto forem diferentes.

## <a name="syntax"></a>Sintaxe

> **.ERRDIF**\[**I**] *textitem1* __,__ *textitem2* \[ __,__ *message*]

## <a name="remarks"></a>Comentários

Se  **&period;ERRDIFI** é fornecido, a comparação não diferencia maiusculas de minúsculas.
