# <a name="ifdif-ifdifi"></a>IFDIF, IFDIFI

Conjunto de concessões se os itens de texto forem diferentes.

## <a name="syntax"></a>Sintaxe

> **IFDIF**\[**I**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Comentários

Se **IFDIFI** é fornecido, a comparação não diferencia maiusculas de minúsculas. Ver [IF](../../assembler/masm/if-masm.md) para a sintaxe completa.
