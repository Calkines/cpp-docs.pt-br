# <a name="record-masm"></a>RECORD (MASM)

Declara um tipo de registro consiste em campos especificados. *FieldName* nomeia o campo *largura* Especifica o número de bits, e *expressão* fornece seu valor inicial.

## <a name="syntax"></a>Sintaxe

> *recordname* RECORD *fieldname*:*width* [[= *expression*]] [[, *fieldname*:*width* [[= *expression*]] ]] ...
