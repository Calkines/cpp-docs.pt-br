# <a name="microsoftwrl-namespace"></a>Namespace Microsoft::WRL

Define os tipos fundamentais que compõem a biblioteca de modelos C++ do Windows Runtime.

## <a name="syntax"></a>Sintaxe

```cpp
namespace Microsoft::WRL;
```
