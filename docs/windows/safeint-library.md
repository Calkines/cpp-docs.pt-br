# <a name="safeint-library"></a>Biblioteca de SafeInt

O **SafeInt** biblioteca de ajuda a impedir estouros de inteiro que podem resultar quando o aplicativo executa operações matemáticas. A versão mais recente dessa biblioteca está localizada em [ https://github.com/dcleblanc/SafeInt ](https://github.com/dcleblanc/SafeInt).
