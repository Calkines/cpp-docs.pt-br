# <a name="make-function"></a>Função Make

Inicializa a classe de tempo de execução do Windows especificada. Use esta função para criar uma instância de um componente que é definido no mesmo módulo.

   typename TArg8,
   typename TArg9

   TArg1 &&arg1,
   TArg2 &&arg2,
   TArg3 &&arg3,

template <
