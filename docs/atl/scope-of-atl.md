# <a name="scope-of-atl"></a>Escopo da ATL

ATL permite que você crie facilmente objetos COM, servidores de automação e controles ActiveX. ATL fornece suporte interno para muitas das interfaces COM fundamentais.

ATL é fornecida como código-fonte que você incluir em seu aplicativo. ATL também faz uma DLL disponível (atl90.dll), que contém o código que pode ser compartilhado entre componentes. No entanto, essa DLL não é necessário.
