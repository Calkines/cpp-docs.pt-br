# <a name="data-transfer-classes"></a>Classes de transferência de dados

As classes a seguir dão suporte a vários tipos de transferência de dados:

- [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) dá suporte à transferência uniforme de dados usando os formatos padrão para recuperar e definir os dados. Notificações de alteração de identificadores de dados por meio do gerenciamento de conexões para informar os coletores.

- [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md) permite que um moniker assíncrono enviar e receber informações sobre a transferência de dados assíncrona de e para seu objeto.
