# <a name="error-information-classes"></a>Classes de informações de erro

A classe a seguir indica como as informações de erro são tratadas:

- [ISupportErrorInfoImpl](../atl/reference/isupporterrorinfoimpl-class.md) Determina se o objeto dá suporte à interface [IErrorInfo](/windows/win32/api/oaidl/nn-oaidl-ierrorinfo) . `IErrorInfo`permite que informações de erro sejam propagadas de volta para o cliente.
