# <a name="building-and-testing-the-atl-project"></a>Compilar e testar o projeto da ATL

Conforme mencionado na [inserindo um controle composto](../atl/inserting-a-composite-control.md), um dos componentes do projeto inicias é uma página HTML padrão que hospeda o controle de composição nova. Depois que você terminar de modificar o controle de composição, clique em **compilar solução** ou **recompilar solução** do **Build** menu. Depois que o projeto é compilado com êxito, carregue a página HTML, localizada no diretório raiz do seu projeto de controle de composição, no Internet Explorer ou outro navegador e testar a funcionalidade do seu controle.

Você também pode testar o controle de composição usando a ferramenta de contêiner de teste, ou qualquer outro aplicativo que pode hospedar um controle ActiveX. Ver [testando propriedades e eventos com contêiner de teste](../mfc/testing-properties-and-events-with-test-container.md) para obter informações sobre como acessar o contêiner de teste.
